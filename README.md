### Installing via Smithery

To install gitingest-mcp for Claude Desktop automatically via [Smithery](https://smithery.ai/server/@puravparab/gitingest-mcp):

```bash
npx -y @smithery/cli install @puravparab/gitingest-mcp --client claude
```

Run mcp inspector
```
uv run mcp dev server.py
```
